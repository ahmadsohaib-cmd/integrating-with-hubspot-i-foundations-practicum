# Integrating With HubSpot I: Foundations Practicum

This repository is for the Integrating With HubSpot I: Foundations course.  
This practicum is one of two requirements for receiving your certification.  
You must also take the exam and score at least 75%.

## Custom Object URL  
https://app.hubspot.com/object-type-settings/50568581/object/2-54067910

## Notes  
Do not add your private app token to this repository.  
A private app was created in the HubSpot account for API access.  
The project uses Express, Axios, Node, and Pug templates.

## Requirements Completed  
- One new homepage route (`/`)  
- One form route (`/update-cobj`)  
- One POST route to create a new record  
- One new Pug template for the homepage  
- Linked custom object in README  
- Private app token stored in `.env`
